# 前言

Shoka 主题默认在进行标签切换时，会将标题修改为 (´Д｀)大変だ！ ，同时修改图标，虽然挺可爱 hh，但是当标签页多了想来回切换时不方便定位。

为此，对 Shoka 主题进行了小的修改：

  1. 切换标签时不再将标题修改为 (´Д｀)大変だ！ ，同时修改图标也是不必要的了；
  2. 保留切换回来时显示的 （●´3｀●）やれやれだぜ 。
